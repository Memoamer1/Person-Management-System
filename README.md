# C# Console Applications (OOP)

A collection of C# console applications focused on practicing and demonstrating
Object-Oriented Programming (OOP) concepts for backend development.

## Technologies Used
- C#
- .NET
- Object-Oriented Programming (OOP)

## Features
- Encapsulation using classes and properties
- Inheritance and polymorphism in practical examples
- Clean code structure and separation of concerns
- User input handling and validation

## How to Run
1. Clone the repository
2. Open the project in Visual Studio
3. Build and run the application

## Purpose
This project was developed to strengthen backend fundamentals and prepare for .NET roles.
